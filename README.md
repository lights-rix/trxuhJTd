# 项目简介

## 核心功能
- 基于YOLOv8的铝型材表面缺陷检测，支持识别10类缺陷：凹陷、不导电、划花、橘皮、漏底、碰伤、气孔、涂层开裂、涂粉、脏点
- 提供多种输入方式：单张图片、图片文件夹、视频文件、本地摄像头
- 支持自定义检测参数：置信度阈值、IOU阈值
- 界面实时显示检测结果：目标位置、目标总数、置信度信息
- 支持批量检测功能，可查看每一张图片的检测结果
- 提供检测结果保存功能

## 适用场景
- 铝型材生产线的质量检测
- 工业制品表面缺陷自动化识别
- 制造业质量控制系统
- 工业图像处理与分析

## 技术特点
- 采用YOLOv8深度学习模型进行目标检测
- 基于Python3.9开发，使用PyQt5构建用户界面
- 集成OpenCV进行图像处理
- 支持PyCharm和VSCode开发环境
- 提供1700张标注数据集用于模型训练
